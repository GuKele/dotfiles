# Dotfiles

my dotfiles for Unbuntu or Debian about zsh && tools && gitconfig.

## Setup

```bash
git clone <repository_url> /your/path
source /your/path/setup.sh
```

## describe

setup.sh will backup your dotfiles in the same directory.
